# escaperoom

#1. 프로젝트 개요

HTML,CSS,JavaScript(jQuery)를 이용하여 웹문서의 기본구조를 이해하고 JSP, Servlet, DB를 이용하여 웹문서에서의 기본 CRUD를 이해하는지 확인

#2. 프로젝트 사용환경

1) Apache-Tomcat 8.5
2) javac 1.8.0_362
3) OpenJDK 1.8.0.345
4) ORACLE 11g
  

#3. 프로젝트 사용 기술

Java, JSP, Servlet, Html, Css, JavaScript(JQuery)

#4. 프로젝트 구현 페이지

1) 로그인
2) 회원가입
3) 게시판
